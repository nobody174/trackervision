# TrackerVision

A modern entity tracking and visualization framework for NeoForge 1.21.1.

TrackerVision replaces vanilla glowing with a flexible target-acquisition and
rendering system: crisp outlines, distance-aware HUD indicators, beacon
tracking, search mode, and target locking.

## Status

Pre-release, active development (v0.1.0 MVP in progress).

## Planned Features (v0.1 MVP)

- `/track lock <target>`, `/track clear`, `/track status` client commands
- Single-target lock
- Crisp outline renderer for the locked target

See [ROADMAP.md](ROADMAP.md) for the full release plan and
[ARCHITECTURE.md](ARCHITECTURE.md) for system design.

## Installation

Not yet released. Build from source with `./gradlew build` (Java 21, NeoForge
21.1.233, Minecraft 1.21.1).
